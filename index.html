ZipFinder



An interactive map for exploring
historical growth and predicted
trends in U.S. zip codes.


Checkboxes

Growth - Top High Growth Zip Codes

Volatility - Top High Volatility Zip Codes

Retailer - Zip Codes with Walmarts

Filter Buttons

High Density - Greater than 5000 people/mi2

Moderate Density - Between 1000 and 5000 people/mi2

Low Density - Less than 1000 people/mi2


Directions

Select the data layers you want to see
using the checkboxes. Then further filter the
data by population density using the grey buttons.
Explore the map to see the results. Finally, click
plot to see the results on the map. Use the
"AI Data Filter" button to alternatively for a less
manual experience.
